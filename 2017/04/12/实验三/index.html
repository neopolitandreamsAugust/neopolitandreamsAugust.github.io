实验三 使用Github管理Android项目

一、实验目的

  • 1.掌握Android Studio中Git的使用
  • 2.掌握Github管理项目

    二、实验任务

  • 1.在Android Studio中新建
  • 2.添加VCS中的Git支持
  • 3.发布项目到Github
  • 4.在Android Studio中进行修改代码、提交、新建分支、合并等操作
  • 5.推送项目到Github
  • 6.如有多人合作，处理pull request

    三、实验步骤与结果

  • 1.框架的构建，根据MC的logo我将新项目命名为Magic Cook
  • 2.VCS -> Import into Version Control -> Share ProjectonGithub
  • 3.输入密码
  • 4.完成分享
  • 5.发布完成后的Github界面

    四、实验小结

  • 一开始我总是输错密码出现Unauthorized bad credentials，导致分享失败，找到原因以后分享很顺利，群组合作使用pull request很方便。
